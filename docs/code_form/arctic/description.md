# La machine code-formation Arctic

Parmi les entités de service « Code formation » de MesoNET, Arctic est matériellement hébergée par le cluster HPC régional (Austral) du mésocentre de Normandie, Criann.

Ses sous-ensembles d'architecture sont décrits sur :
https://services.criann.fr/services/hpc/cluster-austral/architecture/

Arctic est accessible aux projets de développement d’applications ou d’enseignement, en particulier :

- pour des projets d’enseignement du domaine IA sur GPU, des ressources NVIDIA A100 sont attribuées ;

- pour des projets de développement de modèle d’IA sur GPU, des ressources GPU NVIDIA A100 ou H200 peuvent être attribuées ;

- pour des projets HPC (simulation), d'enseignement ou de développement de code, des ressources CPU (AMD Genoa), GPU AMD MI210, GPU NVIDIA A100 ou H200 peuvent être attribuées en fonction des caractéristiques des applications.
